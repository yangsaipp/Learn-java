## 总结

泛型的具体类型只有在使用的时候才知道，故无法通过定义泛型的Class对象直接获取，而对应三种使用场景中可以通过不同方式获取泛型的信息。

获取泛型的类型的大概步骤如下：
1. 首先需要获取Type。
2. 转换为ParameterizedType等Type的子类。
3. 得到泛型类型Class对象。
